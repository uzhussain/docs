# Timekeepur Labs documentation

Docs for the Timekeepur Labs agentic software factory.

## Development

Install the Mintlify CLI to preview the docs locally:

```bash
npm i -g mint
```

Run the dev server at the root where `docs.json` is located:

```
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing changes

Push changes to the default branch to deploy the documentation.
